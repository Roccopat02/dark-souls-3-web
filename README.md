⚔️ Dark Souls III Web

Sitio web temático inspirado en Dark Souls III, desarrollado como trabajo práctico para la materia de Programación Web. El proyecto busca presentar información relevante sobre el juego mediante una interfaz atractiva, navegación intuitiva y diseño responsive.

🌐 Demo

GitHub Pages:
https://roccopat02.github.io/dark-souls-3-web/

📖 Acerca del proyecto

Dark Souls III es un videojuego de rol y acción desarrollado por FromSoftware. Este sitio web fue creado con el objetivo de mostrar información sobre su historia, personajes, localizaciones y elementos más representativos, aplicando conceptos fundamentales del desarrollo web.

✨ Funcionalidades

- 🏠 Página principal con presentación del juego.
- 📜 Sección dedicada a la historia y el lore.
- 👥 Información sobre personajes importantes.
- 🖼️ Galería de imágenes interactiva.
- 📧 Formulario de contacto.
- 📱 Diseño responsive para diferentes dispositivos.
- 📂 Menú de navegación con opciones desplegables.

 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Git y GitHub
- GitHub Pages

🎯 Objetivos del proyecto

- Aplicar conceptos de estructura web con HTML.
- Implementar estilos y diseño responsive con CSS.
- Incorporar interactividad mediante JavaScript.
- Gestionar versiones utilizando Git y GitHub.
- Publicar un sitio web funcional mediante GitHub Pages.
